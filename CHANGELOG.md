## [2.0.0] - 2024

### Added ✨
- **React 18** support with new `createRoot` API
- Modern React patterns and best practices throughout all exercises
- Enhanced educational comments explaining key concepts
- Comprehensive testing examples in `testing/` directory
- GitHub Actions CI/CD pipeline for automated building and testing
- ES6+ syntax adoption across all exercises

### Changed 🔄
- **BREAKING**: Migrated from React 17 to React 18
- **BREAKING**: Replaced deprecated `ReactDOM.render` with `createRoot`
- Updated all dependencies with security fixes
- Modernized component patterns (functional components with hooks)
- Improved repository structure and documentation
- Enhanced README.md with better learning path guidance

### Improved 📈
- Better error handling and user experience
- More comprehensive exercise coverage
- Clearer separation between starter and solution code
- Enhanced testing infrastructure and examples

## [1.0.0] - 2022

### Added 🎯
- Initial repository structure with comprehensive React exercises
- **Fundamentals** section with JSX expressions exercises
- **State Management** exercises covering props and useState
- **Forms** exercises with controlled components
- **Hooks** exercises covering useState and useEffect
- **Redux** exercises with pure Redux and React integration
- **Testing** examples for React and Redux applications
- MIT License
- Comprehensive README with getting started guide

### Organized Exercise Topics 📚
- `fundamentals/jsx-expressions/` - Basic React and JSX concepts
- `state-management/` - Props, state, and component communication
- `forms/` - User input handling with controlled components  
- `hooks/` - React hooks and side effects management
- `redux/` - Complex state management with Redux
- `testing/` - Testing React and Redux applications

### Features 🚀
- Independent React applications for each exercise
- Starter and Solution folders for guided learning
- Create React App based setup for easy development
- Educational comments and documentation throughout
